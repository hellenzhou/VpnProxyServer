# 🚀 极简VPN代理服务器

## 📋 单文件架构

**整个项目只有1个文件：`vpn_proxy_minimal.cpp`**

## 🎯 核心功能

```cpp
// 启动代理
StartProxy(8888);

// 停止代理  
StopProxy();
```

## 🔄 工作流程

```
VPN客户端 → 代理服务器(8888) → 外网服务器
     ↑                              ↓
     ←────── 响应回发 ←──────────────
```

## ✨ 极简特点

- **单文件** - 所有代码在一个文件中
- **无依赖** - 只需要系统库和hilog
- **零配置** - 直接编译运行
- **100行** - 核心逻辑仅100行代码
